# Book Finder

This app uses the Open Library API to search for books and show the results on the page.

It lets the user:

- search for books by title or keyword
- filter results by year
- save books using localStorage

API used:
https://openlibrary.org/developers/api

Live site:
https://pfernandezdecordova1.github.io/NewsToMe/

What I learned:

- how to use fetch with a real API
- how to work with JSON data
- how to display results with JavaScript
- how to handle loading and error messages
