# NewsToMe

Mini Project 3.

This project is my idea for a personalized news app. The goal is to make something simple where users can look at headlines by category, search for topics, and save articles to read later.

Right now, this repo just has the project proposal and README while I plan everything out.

I plan to build it with HTML, CSS, JavaScript, and NewsAPI.
